# auth-service

Microservicio de autenticación desarrollado con Spring Boot.

## Tecnologías utilizadas
- Java 17
- Spring Boot 3.4.6
- Spring Security
- JWT
- Maven
- MySQL
- Swagger/OpenAPI

## Descripción
Este microservicio gestiona la autenticación y autorización de usuarios mediante JWT. Forma parte de una arquitectura basada en microservicios.
